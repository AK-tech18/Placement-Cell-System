# 🎯 Placement Cell System (PCAS)

A full-stack **Placement Management Portal** built using React, Node.js, and MySQL.  
This system helps students apply for jobs, track applications, and analyze placement data.

---

## 🚀 Live Demo

🌐 Frontend: https://placment-cell-system.netlify.app  
🔗 Backend API: https://placement-cell-system-3.onrender.com

---

## 📌 Features

### 👨‍🎓 Student
- View eligible jobs based on CGPA, branch & skills
- Apply for jobs
- Track application status
- Add & update skills
- Upload resume & get feedback

### 🏢 Recruiter
- Post job openings
- Define eligibility criteria (CGPA, branch, skills)

### 🛠️ Admin
- View all students
- Monitor applications
- Analytics dashboard (charts & stats)

---

## 🧠 Smart Features

- 🔥 Skill-based job filtering
- 📊 Placement analytics dashboard
- 📄 Resume Analyzer (PDF parsing + feedback)
- 🎯 Job eligibility engine

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router
- Chart.js

### Backend
- Node.js
- Express.js
- MySQL (Railway DB)
- Multer (file upload)
- pdf-parse (resume analysis)

### Deployment
- Frontend: Netlify
- Backend: Render
- Database: Railway
